Introduction to HTML - Elements

HTML (HyperText Markup Language) is a markup language, since it marks up every piece of content on a web page and defines its type. It is used to create the skeleton of a website, which needs other programming languages to be adorned (CSS) and to become interactive (JavaScript).

Elements

The smallest building block of the HTML language is called element.
An element is identified by an opening and an enclosing symbol, called tag. They embrace the content of the element.
The tag is enclosed in angle brackets, with a specific name that mimics its purpose.
An element can be containing another element, depending on their compatibility.

Paragraph

There are different types of elements. The mostly used is the paragraph, tagged by "p".

Body

Another important element is "body", which is where any content that we want to display to the user will be held. There can be only one body element, such is if it was main function of a script in cpp or python.

Headings

The paragraph of a document is usually anticipated by a title or a subtitle. This elements are called headings.
You can have different heading levels, tagged as "hn" where n is a natural number regarding the hierarchy.
The value of n starts from 1 for the main title and increases for the sections, subsections, etc... reaching its maximum at 6

Line break

In a paragraph you might want to andare a capo without starting another element of the same type.
To do so, you need to use a line breaker, that is another element, tagged as "br".
The just mentioned tag doesn't need to be closed, bcause its a self-closing tag.

Comments

You might want to take notes on your programming interface about what you're writing, without showing the result in the effective HTML file.
This techique is always used by programmers in other programming languages too and is called "commenting". Each note is called "comment".
Each programming language has a specific grammar to write a comment, in HTML you should use the opening elements "<" followed by "!--" and to close just write "-->".

Lists

It usually happens that you need to write a list of elements. In HTML there are two types of lists.
One desplays each single item in the exact order you have written them with a dot, it's tagged as "ul" (unordered list).
The other one gives a natural number to every element, tagging the list as "ol" (ordered list).
Both these lists are elements for HTML and distinguish each item by the tag "li" (listed item).

Ingredients:

  • 1 sleeve (about 29) buttery round crackers such as Ritz, crushed to fine crumbs;
  • 1 cup barbecue sauce, divided;
  • 2 large eggs;
  • 1 teaspoon salt;
  • 1/2 teaspoon freshly ground black pepper;
  • 2 pounds ground beef, 10 to 20% fat;
  • 1 medium sweet onion, grated on the large holes of a box grater (about 3/4 cup).

  2. Put the meatloaf mixture in the middle of the prepared baking sheet and form it into a 9 x 5-inch loaf. Bake the meatloaf for 30 minutes.
  3. Spread the remaining 1/2 cup of barbecue sauce on top of the partially-baked meatloaf, then continue to bake until the center of the meatloaf reads 160Â°F on an instant-read thermometer, 25 to 35 more minutes.
  4. Let the meatloaf cool on the baking sheet for 5 minutes so it slices more neatly. Use a sharp knife to cut the meatloaf into thick slices and serve.
  5. Store leftovers in an airtight container for up to 5 days. Reheat slices of meatloaf in the microwave, covered with a paper towel, until heated through.
